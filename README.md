# Klub Penerjemah Harumodoki

Theme WordPress klasik bergaya website fansub anime era lama.

Theme ini dibuat dengan pendekatan:
- ringan
- sederhana
- nostalgik
- tanpa framework modern
- fokus pada nuansa personal fansub blog

Cocok untuk:
- website fansub
- blog anime
- release archive
- komunitas anime kecil
- website bergaya WordPress klasik

---

## Download Box Shortcode

Gunakan shortcode berikut untuk membuat download section ala fansub klasik.

Contoh:

```txt
[download]
720p|Google Drive|https://example.com
720p|Mega|https://example.com
1080p|Google Drive|https://example.com
[/download]
```

# Instalasi

## 1. Clone repository

```bash
git clone https://github.com/username/klub-penerjemah-harumodoki.git
```

---

## 2. Pindahkan folder theme ke:

```txt
wp-content/themes/
```

---

## 3. Aktifkan theme

Masuk ke:

```txt
Dashboard WordPress → Appearance → Themes
```

lalu aktifkan:
```txt
Klub Penerjemah Harumodoki
```
